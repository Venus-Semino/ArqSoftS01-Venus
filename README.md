# Mi Catálogo de Películas

Este es un proyecto escolar que desarrollé para la materia de arquitectura de software. La idea principal es tener una página donde pueda ver y gestionar mi lista de películas.

##  Tecnologías usadas

Para este proyecto utilicé herramientas modernas que nos enseñaron en clase:

* **Lenguaje:** C#
* **Framework:** .NET 8.0 (ASP.NET Core MVC)
* **Diseño:** Bootstrap 5 y CSS personalizado
* **Scripting:** JavaScript para pequeñas interacciones y efectos de carga.
